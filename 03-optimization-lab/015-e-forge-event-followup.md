# SOP: E Forge Event Follow Up and Awards
**Category:** Optimization Lab | **Duration:** 2h 11m

## Objective
Maximize the authority value of events, awards, and industry recognition — convert participation into long-term AI visibility signals.

## Prerequisites
- Event attended or award received
- PR and brand mention process established (007)

## Step-by-Step Process

### Pre-Event Authority Prep
1. Before event: create event schema on website with Event schema
2. Prepare press kit: logo, bio, headshot, talking points
3. Set up Google Alert + social monitoring for event hashtag

### During Event
4. Post live updates on X/LinkedIn with event hashtag
5. Tag event accounts and speakers in posts
6. Engage with event content — comment, share, repost
7. Collect photos, videos, quotes for post-event content

### Post-Event Authority Maximization
8. **Press Release**: announce participation/award win
   - Include: what happened, why it matters, quote, next step
   - Distribute via tier 1-2 press release networks
9. **Content Creation**:
   - Write recap blog post (800+ words)
   - Create video recap (YouTube, transcribed)
   - Quote event insights in subject matter content
10. **Website Updates**:
    - Add award badge to homepage
    - Create "Awards & Recognition" page
    - Link to event coverage from about page
    - Add press mentions section
11. **Schema Updates**:
    - Add Award schema to recognition page
    - Add Event schema for past events
    - Link from Organization schema to award

### Long-Term Signals
12. Add event/award to Wikidata as statement with reference
13. Update Wikipedia if applicable
14. Feature in future sales decks and proposals
15. Set up follow-up actions from event connections

## Key Deliverables
- [ ] Press release distributed
- [ ] Recap blog + video created
- [ ] Award/recognition page live with schema
- [ ] Wikidata updated with event/award

## Verification
Award appears in Google Knowledge Panel (if applicable)
Event/award generates 3+ new brand mentions
Website shows award/recognition prominently
