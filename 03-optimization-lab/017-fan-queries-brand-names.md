# SOP: Fan Queries and Brand Names
**Category:** Optimization Lab | **Duration:** 1h 38m

## Objective
Identify and optimize for "fan queries" — the long-tail, conversational, high-intent search queries your audience actually uses — and protect/manage brand name variations in AI search.

## Prerequisites
- Search Console access
- Keyword research tool (Semrush, Ahrefs, or similar)

## Step-by-Step Process

### Fan Query Discovery
1. **Mine Search Console**: filter queries with impressions but low CTR → these are opportunity queries
2. **Mine "People Also Ask"**: scrape PAA boxes for your main keywords
3. **Client interview**: ask top 5 clients "what did you search to find us?"
4. **Social listening**: check Reddit, Quora, X for how people ask about your services
5. **AI trend analysis**: query ChatGPT/Gemini "what questions do people ask about [service]"

### Fan Query Categorization
6. Organize queries by intent:
   - **Informational**: "how does [service] work?"
   - **Comparison**: "[service A] vs [service B]"
   - **Local**: "[service] near me"
   - **Problem**: "why is my [problem] happening?"
   - **Cost**: "how much does [service] cost?"
7. Map queries to content types: FAQ, blog post, video, comparison page

### Brand Name Protection
8. Audit brand name variations:
   - Common misspellings (check Search Console queries)
   - Abbreviations
   - Hyphenated versions
   - "Old name" if rebranded
9. Register brand name variations as domains (301 redirect to main)
10. Create content optimized for misspelled brand queries
11. Monitor for brand impersonation on social platforms
12. Check AI responses for brand name accuracy across engines

### Content Optimization for Fan Queries
13. Create conversational content: Q&A format, question in H1/H2
14. Use natural language throughout (match query phrasing)
15. Implement FAQ schema targeting these specific queries
16. Create "quick answer" paragraph at top of optimized pages

## Key Deliverables
- [ ] Fan query list (50+ queries by category)
- [ ] Content mapped to each query type
- [ ] Brand name variation audit complete
- [ ] Misspelled brand domains registered

## Verification
Fan query pages appearing in featured snippets
Search Console shows improved CTR on targeted queries
AI search engines correctly reference brand name
