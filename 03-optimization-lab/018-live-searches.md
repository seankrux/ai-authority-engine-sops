# SOP: Live Searches
**Category:** Optimization Lab | **Duration:** 1h 19m

## Objective
Conduct and analyze live AI search tests — real-time queries across multiple AI engines to audit current visibility, identify gaps, and track optimization progress.

## Prerequisites
- List of target keywords and queries
- Access to multiple AI search platforms

## Step-by-Step Process

### Live Search Setup
1. Prepare query list (10-20 per session):
   - Brand queries: "[brand name]", "[brand] [service]"
   - Category queries: "[service] [city]", "best [service]"
   - Problem queries: "[problem] fix", "how to [solve]"
   - Comparison queries: "[brand] vs [competitor]"
   - Voice/long-tail: "hey google, find me [service]"
2. Set up recording method: screenshot tool or screen recorder
3. Clean browser cache before each test (avoid personalization)

### Multi-Engine Query Run
4. Query each keyword across:
   - **Google** (web + featured snippets)
   - **Google AI Overviews** (SGE)
   - **ChatGPT** (with and without web search)
   - **Perplexity** (pro search + focus modes)
   - **Gemini** (web-enabled)
   - **Claude** (with web search)
   - **Copilot** (Microsoft)
   - **Grok** (X integration)
5. Document per result:
   - Did your brand appear? (Y/N)
   - Position/placement (top 3, top 5, beyond)
   - Accuracy of brand description (accurate, partial, wrong)
   - Competitors mentioned
   - Type of result (direct answer, list, suggestion, citation)

### Analysis & Action
6. Calculate AI Visibility Score: (appearances / total queries) × 100
7. Compare against previous baseline
8. Identify consistency gaps: "appears in ChatGPT but not Perplexity"
9. Flag inaccurate AI descriptions → correct via source data (GBP, Wikidata, schema)
10. Prioritize fixes based on highest-volume underperforming queries

### Ongoing Monitoring
11. Schedule bi-weekly live search sessions
12. Maintain query list — add new, retire irrelevant
13. Track visibility score trend weekly
14. Alert when competitor appears where you don't

## Key Deliverables
- [ ] Live search session results documented
- [ ] AI Visibility Score calculated
- [ ] Accuracy issues flagged and sourced
- [ ] Priority optimization actions identified

## Verification
AI Visibility Score trending upward weekly
Accuracy issues resolved within 1 week
Brand consistently appears across 5+ engines for primary queries
