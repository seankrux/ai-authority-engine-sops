# SOP: Building Reviews, Reputation & AI Trust Signals
**Category:** Getting Started — Lesson 3 | **Duration:** 1h 27m

## Objective
Build a robust review generation system and reputation management pipeline that feeds positive trust signals into AI ranking algorithms.

## Prerequisites
- AI Presence Layer established (003)
- Google Business Profile claimed and verified

## Step-by-Step Process

### Review Generation System
1. Audit current review status: Google, Yelp, Facebook, industry-specific (Avvo, Healthgrades, etc.)
2. Create review funnel: SMS/email → review link → follow-up sequence
3. Set up automated review requests 24h after service completion
   - Use: Widewail, BirdEye, Podium, or manual template
   - Cutoff: don't request after 7 days (diminishing response)
4. Create response templates for 5-star, 3-4 star, and <3 star reviews

### Reputation Management
5. Respond to ALL reviews within 48 hours — Google prioritizes active profiles
6. For negative reviews: acknowledge publicly, offer resolution path, take offline
7. Generate 5+ new reviews per month minimum (target: 10+)
8. Feature best reviews as testimonials on website + schema

### AI Trust Signal Deployment
9. Implement AggregateRating schema on homepage
10. Add Review schema to testimonial pages
11. Create trust badges page with industry certifications
12. Link to authoritative third-party reviews from your site

### Monitoring & Reporting
13. Set up weekly review digest (new reviews, response status)
14. Track review velocity: reviews/month target
15. Quarterly reputation audit — check AI responses for review sentiment

## Key Deliverables
- [ ] Automated review request workflow
- [ ] Response templates for all star ratings
- [ ] AggregateRating + Review schema deployed
- [ ] Review velocity tracking dashboard

## Verification
Google Business Profile shows 4.5+ stars with 20+ reviews
AI search results include positive review snippets
Review response rate = 100% within 48h
