# SOP: Building Out Wikidata and Grokipedia
**Category:** Getting Started — Bonus 2 | **Duration:** 1h 43m

## Objective
Establish and optimize your brand's presence on Wikidata (the structured knowledge base behind Wikipedia and Google's Knowledge Graph) and Grokipedia (xAI/Grok's knowledge source).

## Prerequisites
- Organization schema live
- Website with complete About page
- Press mentions or credible third-party sources

## Step-by-Step Process

### Part A: Wikidata Setup

#### If No Entry Exists — Create One
1. Go to https://www.wikidata.org → Create account
2. Click "Create a new Item"
3. Fill in:
   - Label: Business name
   - Description: One-line business description
   - Aliases: Common variations, former names
4. Add statements (properties):
   - `P31 (instance of)`: business, organization, or specific type
   - `P17 (country)`: your country
   - `P131 (located in)`: city
   - `P159 (headquarters location)`: address
   - `P856 (official website)`: URL
   - `P1451 (motto text)`: tagline
   - `P452 (industry)`: your industry
   - `P112 (founded by)`: founder name(s)
   - `P571 (inception)`: founding date
   - `P749 (parent organization)`: if applicable
5. Add references for each statement (website, press articles, official records)
6. Add `P2002 (X/Twitter)` and other social profile IDs

#### If Entry Exists — Improve It
7. Search your brand → note missing/incorrect statements
8. Add missing properties with references
9. Correct any outdated information
10. Add sitelinks to Wikipedia (if Wikipedia page exists)

### Part B: Grokipedia Setup
11. Search Grok/xAI: query your brand + industry
12. Note what Grok "knows" about your brand
13. To influence Grok:
    - Create/update X/Twitter profile with complete bio and link
    - Post consistent content about your expertise
    - Get mentioned by authoritative X accounts
    - Link your website from X profile
14. Verify Grok responses improve over 30 days as signals accumulate

### Part C: Cross-Linking
15. Link Wikidata entry to website (via schema + footer)
16. Connect Wikidata to Wikipedia (if applicable)
17. Ensure Google Knowledge Panel references Wikidata as source
18. Use Wikidata as the single source of truth for all citation data

## Key Deliverables
- [ ] Wikidata entry created/optimized with 10+ statements
- [ ] All statements have verifiable references
- [ ] X/Grok profile optimized for Grokipedia
- [ ] Knowledge Panel referencing Wikidata

## Verification
Google Knowledge Panel matches Wikidata data exactly
Grok search returns accurate brand description
Wikidata statements verified and accepted
