# SOP: Building Optimized Pages for AI Visibility
**Category:** Getting Started — Lesson 4 | **Duration:** 2h 10m

## Objective
Create and optimize website pages specifically engineered for AI answer engine indexing, featured snippet capture, and generative search visibility.

## Prerequisites
- AI Trust Signals established (004)
- Core Web Vitals passing on desktop + mobile

## Step-by-Step Process

### Page Architecture for AI
1. Audit current pages: identify AI-visible vs AI-invisible pages
2. Structure pages for AI consumption:
   - Clear H1 with primary keyword (1 per page)
   - Logical H2/H3 hierarchy (answers sub-questions)
   - Direct answer paragraphs (first 40-60 words = the answer)
   - FAQ sections at bottom (target "People Also Ask" entries)
3. Implement "question → answer" format for key search queries

### Content Optimization
4. Write for featured snippets:
   - Define terms concisely (paragraph snippet)
   - Use numbered steps (list snippet)
   - Include data tables (table snippet)
   - Add bullet-point breakdowns
5. Include "word from our brand" authority statements
6. Link internally using descriptive anchor text (not "click here")

### Technical Optimization
7. Ensure all pages are indexable (no noindex, robots.txt allowing)
8. Add breadcrumb navigation with BreadcrumbList schema
9. Implement canonical URLs correctly
10. Set up XML sitemaps with priority flags for AI-optimized pages
11. Optimize page speed: under 2.5s LCP for key pages

### AI-Specific Page Types
12. Create "vs" pages (Brand vs Competitor) for AI comparison queries
13. Create "near me" optimized local pages
14. Build comprehensive pillar pages (2500+ words with internal links)
15. Add "as seen in" / "featured in" trust section

## Key Deliverables
- [ ] 5+ pages optimized for AI snippet capture
- [ ] FAQ schema on all AI-target pages
- [ ] BreadcrumbList schema deployed site-wide
- [ ] Featured snippet opportunity map complete

## Verification
Pages appear in top 3 positions for featured snippets
AI answer engines cite your content in responses
"People Also Ask" section includes your pages
